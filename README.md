# V2X Traffic Safety System Simulation

## Overview
A Python simulation comparing vehicle safety at a 4-way intersection **with** and **without** V2X (Vehicle-to-Everything) communication. Demonstrates how DSRC-based V2V and V2I communication catches dangers that physical sensors alone miss — especially at blind intersections.

## Key Concepts
- **DSRC** (Dedicated Short-Range Communications) — IEEE 802.11p, 5.9 GHz
- **V2V** (Vehicle-to-Vehicle) — BSM (Basic Safety Message) broadcasting
- **V2I** (Vehicle-to-Infrastructure) — SPaT (Signal Phase and Timing) from traffic lights
- **Multi-Agent Communication** — Independent agents sharing state for collective safety
- **TTC** (Time-to-Collision) — Primary safety metric
